To instantiate virtual environment:---
source .venv/bin/activate

to get all requirements:---
pip install -r requirements.txt

Run main.py to start the website, it will give you your url in the terminal 
debug mode = on in main.py at the end allows for webste to update on refresh 
when code is changed
If its not working it could be from browser caching, chrome works for me but safari 
caches a lot.
